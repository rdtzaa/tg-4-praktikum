## 1. Knights Tour
Cara menggunakan program:
1. Gitclone repository ini kedalam local / pindahkan kedalam collab.
2. Jalankan program dari code block pertama.
3. Masukkan input untuk x dan y, titik awal knight untuk menyelesaikan tour (titik 0,0 berada di kiri bawah).
4. program berjalan.
